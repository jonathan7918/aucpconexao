# AUCP — Site estático

Site da Associação Universal Conexão Profunda em HTML/CSS/JS puro.

## Publicar no GitHub Pages
1. Cria um repositório novo no GitHub.
2. Faz upload de todos os ficheiros desta pasta.
3. Em **Settings → Pages**, escolhe a branch `main` e a pasta `/ (root)`.
4. O site fica disponível em `https://<utilizador>.github.io/<repo>/`.

## Estrutura
- `index.html`, `sobre.html`, `servicos.html`, `inscricao.html`, `contacto.html`
- `styles.css`, `script.js`
- `assets/` (logo + hero)

## Formulários
Os formulários abrem o cliente de email do utilizador (mailto) para enviar a candidatura/mensagem para `aucpconexao@gmail.com`.
